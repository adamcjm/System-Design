## API服务框架

对于外部AP I 调用或者客户端对后端API 的访问，可以使用HTTP 协议或者RSTful（当然也可以直接通过最原始的Socket 来调用） 。但对于内部服务间的调用， 一般都是通过RPC 机制来进行的。
